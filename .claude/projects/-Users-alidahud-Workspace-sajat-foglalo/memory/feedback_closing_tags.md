---
name: Include closing tags in edits
description: When changing an HTML/JSX tag name, always include the closing tag in the same edit
type: feedback
---

When changing an HTML/JSX element tag (e.g. h3 to h2), always include both the opening and closing tag in the edit.

**Why:** User had to interrupt and remind me to also change the closing tag.

**How to apply:** Always match opening and closing tags together in a single edit operation.
